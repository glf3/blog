分析：

根据题意来解决即可。

  • 根据题意判断字符串初始条件，处理空字符串的情况。
  • 使用一个指针p1指向haystack（长度为hl）所在位置，另一个指针p2指向needle（长度为nl）所在的位置：
    1. 如果haystack[p1] == needle[p2]并且p1 < hl && p2 < nl，那么p1++，p2++，继续往后匹配。
    2. 反之则退出
      1. 如果p2 == nl的话，说明needle在haystack中全部匹配完成，说明已经找到了。
      2. 否则，把p1++，p2 = 0，代表从haystack的p1位置开始重新匹配needle，继续重复以上步骤即可。
