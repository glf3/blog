Algorithm

本周算法题5. 最长回文子串

题意：

分析：

对于这个问题，可以使用暴力去解决，时间复杂度为O(n^3)，也可以使用动态规划去解决，时间复杂度为O(n^2)。我用动态规划解决的。

  • 定义字符串为str，dp[i][j]代表的意义是从str(i,j)区间内的最长回文子字符串。
  • 初始dp[i][i] = 1，代表单个字符串是回文字符串。
  • 状态转移方程是：
    1. 当str(i) == str(j)的时候，并且str(i+1)(j-1)也为回文串的时候，dp[i][j] = dp[i+1][j-1] + 2
    2. 当str(i) != str(j)的时候，dp[i][j] = max(dp[i+1][j], dp[i][j-1])
    3. 可以证明，当str(i) == str(j)并且str(i+1,j-1)也为回文串的时候，dp[i][j]状态转移一定是从dp[i+1][j-1]转移而来。
  • 定义dp[i][j]为从str(i,j)之间的最大回文子串，那么dp(i+1,j-1)作为子问题，子问题具有最优解，并且没有影响dp[i][j]做的选择，因此具有最有子结构。

Review

本周阅读了一篇Medium上的文章《A Study List for Java Developers》


作者说自己成为Java开发者已经6年了，在学习Java的过程中走了很多弯路，于是总结了一些学习Java的建议清单给新人。

写出很清晰简单的代码

这是推荐的书籍，它们可以让你写出很清晰简洁的代码：

  • Clean Code (2008), by Robert C. Martin
  • Effective Java (2017), third edition, by Joshua Bloch

作者建议在leetcode上挑选100道简单的问题，然后自己把他们全部解决，然后再和去和最好、代码最少的解法相比较，发现两者的不同，内化你的学习，然后通过每一道习题5遍以上。最终你会发现你会写出越来越简洁的代码。

掌握语言还有JDK

作者建议大家都可能学会了基本的语法，但是没有跟过Java官方的学习指南，建议大家都能跟着学一遍《official Java tutorials》，并且学习如下所示比较专业的知识：

  • Custom Networking
  • Generics
  • JavaBeans
  • JDBC Database Access
  • JMX
  • JAXB
  • Reflection
  • Security (a more interesting place for Java Security is actually here, which is referenced in this trail).

推荐一些有用的JDK工具《JDK Tools and Utilities》

以及阅读《Java Language Specification》的第十七章《 Chapter 17 Threads and Locks》

学习Gradle

Gradle成为越来越多的Java项目的编译工具了。关于Gradle的学习导航：

  • 《Gradle User Guide》
  • 《Plugin Development Tutorials》

掌握Jetty

除了工作之外，Jetty对我来说最重要的价值就是服务于我的项目。掌握好了Jetty给了我极大的便利。
下面是学习Jetty的文档：

  • 《The official Jetty doc》

使用Hadoop

在Java中掌握如何写map-reduce(MR) job是一件很有价值的事情，因为你可能通过它深刻地理解一些其他的问题。

熟悉Java EE

推荐阅读JavaEE官方文档+ 《official Java EE tutorial》

  • Part III Chapter 6 Getting Started with Web Applications。特别是刚开始的两个介绍部分。
  • Part III Chapter 17 Java Servlet Technology。人们现在已经很少使用低级的servlet APIs了，大家都使用建立在servlet APIs上的框架，但是熟悉Servlet还是很重要的。
  • Part IV Bean Validation。你可能永远不会直接使用JAVA EE bean验证技术，但它解决的底层问题是通用的。
  • Part V Contexts and Dependency Injection for Java EE。你可能不会直接使用CDI，但是学习它可以帮助你学习依赖注入框架的基本概念。
  • Part VI Web Services。这部分特别适合像我这样的服务器端Java开发人员。
  • Part VIII Persistence。很相关。
  • Part X Security

使用Spring

当你成为服务端开发很长一段时间的时候，Spring framework成为了必然的选择。推荐书籍：

  • 《Spring framework documentation》

领悟Zookeeper

Zookeeper不仅仅是大多数Java开发者需要学习的，它也变成了开发一个分布式系统不可或缺的一部分。推荐书籍：

  • Chapter 21 Zookeeper, in Hadoop: The Definitive Guide (2015), by Tom White
  • 《The official Apache Zookeeper documentation》

总结

这个学习清单严重倾向于服务端的Java开发工程师，因为我也是其中的一员，但它对成为一个好的Java开发工程师不是必须的。成为一个好的Java开发工程师的话需要比上述更加复杂的多的技能清单，比如很强的故障排除能力等。

另外，推荐多上上Stack Overflow问一些好的问题，当你google不到或者学习以上知识有什么疑问的时候就可以去上面问一问。
