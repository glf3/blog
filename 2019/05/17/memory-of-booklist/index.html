关于书单的记忆

发表于 2019-05-17

看了一下自己github的提交记录，偶然发现了有一个是2014年还在大学时的提交，内容是当时自己的阅读书单，不得不说我上大学的时候看书的状态真是纯粹与幸福。

毕业之后的阅读不可避免的被功利、事务、疲劳损害，同样比起大学时期精神上的无限可能性，现在的我精神上却是一个瘦皮球。sad:(
